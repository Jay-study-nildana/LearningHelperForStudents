# Learning Helper For Students

A simple nuget package to share with my students. These contain some common utilities that help me practice C# fundamentals. 

For example, when practising strings and collections, I have to constantly keep creating a collection of strings. RandomRepeatedWordsGenerator solves the problem, saving me (and my students) anywhere from 5 seconds to 50 seconds.

## Current facilities in this package

1. GenerateRandomRepeatedWords : Simple generator that generates a collection of random words with a specified range of frequency
1. DisplayWordsList : Simple utility that returns a string that is joined into a single string
1. AddSimpleConsoleDivider : Adds a simple dash divider in the console output
1. DisplayListOfStringsNow : Simply displays a list of strings
1. GetStringInput : Collect User input.
